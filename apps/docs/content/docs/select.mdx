The `Select` component represents a fundamental UI building block within the DesignForge token system.

## Installation

```bash
npm install @designforge/ui
```

## Usage

```tsx
import { Select } from "@designforge/ui";

export default function App() {
  return (
    <Select>Example Content</Select>
  );
}
```

### Attributes

| Prop      | Type      | Default       | Description                                                 |
| :-------- | :-------- | :------------ | :---------------------------------------------------------- |
| `className` | `string`  | `undefined`   | Merges all optional utility styles directly. |
