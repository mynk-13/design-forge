The `Accordion` component represents a fundamental UI building block within the DesignForge token system.

## Installation

```bash
npm install @designforge/ui
```

## Usage

```tsx
import { Accordion } from "@designforge/ui";

export default function App() {
  return (
    <Accordion>Example Content</Accordion>
  );
}
```

### Attributes

| Prop      | Type      | Default       | Description                                                 |
| :-------- | :-------- | :------------ | :---------------------------------------------------------- |
| `className` | `string`  | `undefined`   | Merges all optional utility styles directly. |
