The `Label` component represents a fundamental UI building block within the DesignForge token system.

## Installation

```bash
npm install @designforge/ui
```

## Usage

```tsx
import { Label } from "@designforge/ui";

export default function App() {
  return (
    <Label>Example Content</Label>
  );
}
```

### Attributes

| Prop      | Type      | Default       | Description                                                 |
| :-------- | :-------- | :------------ | :---------------------------------------------------------- |
| `className` | `string`  | `undefined`   | Merges all optional utility styles directly. |
