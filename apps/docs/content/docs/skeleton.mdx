The `Skeleton` component represents a fundamental UI building block within the DesignForge token system.

## Installation

```bash
npm install @designforge/ui
```

## Usage

```tsx
import { Skeleton } from "@designforge/ui";

export default function App() {
  return (
    <Skeleton>Example Content</Skeleton>
  );
}
```

### Attributes

| Prop      | Type      | Default       | Description                                                 |
| :-------- | :-------- | :------------ | :---------------------------------------------------------- |
| `className` | `string`  | `undefined`   | Merges all optional utility styles directly. |
