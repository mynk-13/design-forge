The `Avatar` component represents a fundamental UI building block within the DesignForge token system.

## Installation

```bash
npm install @designforge/ui
```

## Usage

```tsx
import { Avatar } from "@designforge/ui";

export default function App() {
  return (
    <Avatar>Example Content</Avatar>
  );
}
```

### Attributes

| Prop      | Type      | Default       | Description                                                 |
| :-------- | :-------- | :------------ | :---------------------------------------------------------- |
| `className` | `string`  | `undefined`   | Merges all optional utility styles directly. |
